## 📌 Deskripsi

Tugas ini membahas penggunaan percabangan IF dan CASE dalam MySQL.
Percabangan digunakan untuk menentukan keputusan secara otomatis berdasarkan kondisi tertentu.

2. Jalankan query atau procedure
3. Contoh:
